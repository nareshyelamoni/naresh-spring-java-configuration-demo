# Spring Java-Based Configuration and Dependency Injection Demo

## Overview

This repository demonstrates the implementation of **Java-Based Configuration** in the Spring Framework, eliminating the need for traditional XML configuration files. The project highlights how Spring's IoC Container manages application components, creates beans, and injects dependencies using annotations and configuration classes.

The application generates personalized greetings based on the current system time, providing a simple yet effective example of bean creation, dependency injection, and annotation-driven development.

## Application Flow

1. The Spring Container is initialized using a Java Configuration class.
2. Components are automatically discovered through component scanning.
3. Required beans are created and registered within the IoC Container.
4. Dependencies are injected automatically into application components.
5. The service layer processes business logic and generates greetings based on the current time.
